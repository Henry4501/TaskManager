Task Manager Workspace
A highly polished, Notion-inspired Task Manager built with Vanilla JavaScript, HTML5, and CSS3. This application allows users to manage their daily workflow within a calm, structured dashboard featuring live statistics and a responsive layout.
🚀 Features
Task Composition: Add tasks via the input field by clicking "Add Task" or pressing the Enter key.
Task Management:
Complete Tasks: Mark tasks as finished with a strikethrough effect.
Delete Tasks: Remove tasks instantly from the workspace.
Live Statistics: A sidebar that tracks the total number of tasks and the number of completed items in real-time.
Empty State: Dynamic messaging that appears when the task list is empty to maintain a clean canvas.
Advanced UI:
Glassmorphism: Frosted glass effects using backdrop-filter.
Responsive Design: Fully optimized for Desktop, Tablet, and Mobile views.
Soft Surfaces: A custom color palette designed for a "calm" productivity experience.
🛠️ Technologies Used
HTML5: Semantic structure (aside, main, section, ul, li).
CSS3: Advanced techniques including:
CSS Custom Properties (Variables)
CSS Grid and Flexbox for layout
Glassmorphism & Radial Gradients
Media Queries for responsiveness
JavaScript (ES6+):
DOM Manipulation (Creating and removing elements)
Event Listeners (Click and Keypress)
State Management (Tracking counts)
